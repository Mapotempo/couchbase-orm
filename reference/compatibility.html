Rails Frameworks Support#

Ruby on Rails is comprised of a number of frameworks, which CouchbaseOrm attempts to provide compatibility with wherever possible.

Though CouchbaseOrm attempts to offer API compatibility with Active Record, libraries that depend directly on Active Record may not work as expected when CouchabseOrm is used as a drop-in replacement.

Note

CouchbaseOrm can be used alongside Active Record within the same application without issue.
